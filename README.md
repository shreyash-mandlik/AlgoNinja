# 🥷 AlgoNinja — DSA Learning Platform

Interactive DSA learning platform with algorithms, 
visualizations, quizzes and interview prep!

## 🌐 Live Demo
👉 [Try AlgoNinja!](https://algoninja-ccapcvrtdanrnq2rpmcz5n.streamlit.app/)

## 🎯 Features
- 📚 18+ algorithms with theory, pseudocode and code
- 💻 4 languages — Python, C++, Java, JavaScript
- 🧩 70+ quiz questions
- ⚡ Algorithm Race — compare speeds
- 🎯 40+ interview questions with timed practice
- 📊 Big O complexity visualizer
- 🔍 Search any algorithm instantly

## 📚 Categories Covered
- 🔍 Search: BFS, DFS, Binary Search, A*, Dijkstra, Greedy
- 🔢 Sorting: Bubble, Selection, Insertion, Merge, Quick, Heap
- 🗺️ Graph: Floyd Warshall, Topological Sort
- ⚡ DP: Fibonacci, Knapsack, LCS
- 🎮 Game: Minimax, Alpha-Beta Pruning
- 🤖 Agents: All 5 intelligent agent types

## 🛠️ Tech Stack
- Python, Streamlit
- Pandas, NumPy, Matplotlib
